# EchoChain: Circular Economy & Secondary Market Lifecycle Analytics

## 📌 Project Overview

EchoChain is a Data Analytics project that helps manufacturers analyze the complete lifecycle of electronic products. The project combines manufacturing, warranty, and secondary market data to identify products suitable for refurbishment and buy-back programs.

---

## 🎯 Problem Statement

Manufacturers usually track products only until they are sold. After that, they lose visibility into the product lifecycle.

This project analyzes:

- Manufacturing (BOM) Data
- Warranty Claims
- Secondary Market (eBay) Listings

to calculate a **Circularity Score** that helps identify products with high refurbishment potential.
